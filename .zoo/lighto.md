# Lighto overrides — Longrun

- Ticket tooling: none. Tasks originate in chat; task files under `.spec/`,
  research under `.tasks/` (Lighto defaults).
- Project is generated by XcodeGen from `project.yml`. After editing
  `project.yml` or adding/removing source files, regenerate:
  `xcodegen generate` (the `.xcodeproj` is gitignored).
- Tier-0 / build & test commands:
  - Core unit tests (fast, no Xcode): `swift test --package-path LongrunCore`
  - Build app: `xcodebuild -project Longrun.xcodeproj -scheme Longrun \
    -configuration Debug -destination 'platform=macOS' \
    -derivedDataPath .build/dd -allowProvisioningUpdates -quiet build`
  - Built app path: `.build/dd/Build/Products/Debug/Longrun.app`
  - Signing: automatic, team `D963M2VVCH` (Apple Development cert).
- `.gitignore` exists; ignores Xcode/SwiftPM noise, the generated
  `Longrun.xcodeproj/`, `.spec/`, and `.tasks/`.
