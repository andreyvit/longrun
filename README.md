# Longrun

Keeps your long-running dev commands alive — a macOS app for dev servers, ssh
tunnels, ngrok and the like, with live output and regexp-triggered
notifications.

The point: stop keeping a Terminal tab open for every mundane background tool.
Define each command once, auto-start it on app launch, watch its output in an
embedded terminal, get notified when something interesting (or bad) shows up.

**Status: early development — nothing to run yet.**
